CSS Syntax

Css syntax in general used starting with a selector then defining the property you want to change followed by the value

Selector{
  property: value; }

How to apply CSS

In order to apply CSS to your html file 3 methods can be applied:

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS

Inline CSS is applied by using style attribute inside the element of your html source code
< element style=“property:value”> The previous line is styled using inline css

Internal CSS

Internal CSS is used inside the <head> element of the html page. An element called <style> is used inside the head where the CSS code is written. The head and style element is styled using internal css.

External CSS

This is the most widely used to method to add styling to your Webpage, a seperate syle.css file is used which is linked to the html file in the header using <link>. This page is styled using this method.

Css Selectors
Elemnent

Elements can be used to in stylesheet to change their value and property. They have low Specificity than class and id. They are selected in this method:
Selector{
  property:value; }

Class

Classed can be used to group elements by assigning them a class in their tags and calling them in the syle sheet as :
.classname{ property:value;}
Class is more specific than elements.

Id

Id is the most specific among these three. Id is selective as it is assigned to a single element ,so in general one element is assigned to it's unique id . Id's are selected in styling using the ocathrope/hash symbol:
#idname{property: value;}
